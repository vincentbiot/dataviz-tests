## Typographie

### Titres Display
- **Titre display large - Regular - 48px**
  - Utilisation: Titres principaux de grande taille
  - Poids: Regular (400)

- **Titre display large - Bold - 56px**
  - Utilisation: Titres emphases de très grande taille
  - Poids: Bold (700)

### Titres Hiérarchiques
- **Titre H1 - 40px**
  - Utilisation: Titres de premier niveau

- **Titre H2 - 32px** (visible dans l'image)
  - Utilisation: Titres de section

- **Titre H3 - 24px** (déduit de la hiérarchie)
  - Utilisation: Sous-titres

### Corps de Texte
- **Corps de texte - 16px** (standard)
  - Utilisation: Texte principal, paragraphes

- **Corps de texte alternatif - 14px**
  - Utilisation: Texte secondaire, notes

**Note:** La famille de police exacte n'est pas clairement visible dans le PNG. Elle pourrait être:
- Une police système moderne (Inter, SF Pro, Segoe UI)
- Une police Google Fonts (Roboto, Open Sans)
- Une police personnalisée

## Recommandations d'Utilisation

### Hiérarchie Visuelle
1. Utiliser les titres display (48-56px) pour les headers principaux
2. Utiliser H1 (40px) pour les titres de page
3. Utiliser H2-H3 (32-24px) pour les sections et sous-sections
4. Corps de texte standard (16px) pour le contenu principal
5. Texte alternatif (14px) pour les labels, légendes, notes

### Couleurs
- **Couleurs principales**: Pour les éléments importants (CTA, navigation)
- **Couleurs secondaires**: Pour les éléments complémentaires
- **Couleurs neutres**: Pour le texte, bordures, backgrounds
- **Couleurs d'accent**: Pour attirer l'attention (alertes, badges, highlights)

### Espacement
- Maintenir une grille cohérente basée sur des multiples de 8px
- Utiliser les espacements généreux pour améliorer la lisibilité
